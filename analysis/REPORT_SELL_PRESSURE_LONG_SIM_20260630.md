# sell_pressure時LONG許可シミュレーションレポート

**作成日**: 2026-06-30  
**検証期間**: 2026-06-25 〜 2026-06-30（4営業日）  
**対象銘柄**: 6981（村田製作所）, 6920（レーザーテック）

---

## 1. 背景と目的

現行システムでは、板データが「sell_pressure」を示している場合にLONGエントリーを一律禁止している。しかし6/30のように「売り板が厚いのに買いが突破する」パターンでは、大きな上昇を取り逃す構造的問題がある。

本シミュレーションでは、sell_pressure時でも特定条件下でLONGを許可した場合の損益を検証する。

---

### エントリー条件

| 項目 | 設定値 |

### 時間フィルター

- 09:05前: エントリー禁止

- 12:30〜13:00: エントリー禁止
- 14:30以降: エントリー禁止

## 6. 重要な発見

### 6-1. sell_pressure時のエントリーが利益源

| 板状態 | 件数 | 損益 | 期待値 |
|--------|------|------|--------|
| **sell_pressure中** | **13件** | **+20,260円** | **+1,558円/回** |
| non-sell_pressure | 6件 | -280円 | -47円/回 |

**sell_pressure中の大台超えLONGこそが利益源**であり、非sell_pressure時はほぼ収支均衡。これは「売り板を突破する買い圧力」が存在する場面でのみ、大台超えが意味を持つことを示している。

### 6-2. BEストップの効果

| 項目 | 結果 |
|------|------|
| BE発動回数 | 12件/19件（63%） |
| BE→建値決済 | 9件 |
| BE→TP到達 | 3件 |
| BE未発動→SL | 7件 |

BEストップがなければ、BE建値決済の9件のうち多くがSL(-0.5%)に到達していた可能性が高い。BEストップにより推定 **-44,000円〜-89,000円の損失を回避**している。

### 6-3. 前場 vs 後場

| 時間帯 | 件数 | 勝率 | 損益 |
|--------|------|------|------|
| 前場（〜11:30） | 15件 | 20% | **+49,959円** |
| 後場（12:30〜） | 4件 | 0% | **-29,979円** |

**後場のエントリーは全敗**。前場に限定すれば：
- 純損益: +49,959円
- PF: 2.52
- 期待値: +3,330円/回

## 8. リスク評価

### 下落日（6/26）での安全性

6/26は下落日だが、このロジックで2件エントリーが発生：
- 6981 10:01: BE建値決済（±0円） → BEストップが保護
- 6920 10:16: SL(-9,749円) → 損失発生

**下落日の損失は-9,749円に限定**されており、BEストップが1件を保護している。ただし完全に安全ではない。

### 最大ドローダウン

連続損失の最大は6/25の09:38〜10:31（6920で-9,878円 → BE → BE → -9,994円）= -19,872円。

## 9. 改善提案

### 推奨設定（前場限定版）

| 項目 | 推奨値 |
|------|--------|
| 対象銘柄 | 6981, 6920のみ |
| エントリー時間 | 09:05〜11:00 |
| シグナル | 大台超え + medium以上 |
| 追加条件 | close > VWAP + 直近3本陽線優勢 |
| TP | +1.5% |
| SL | -0.5% |
| BE | +0.5%で建値移動 |
| 1日最大エントリー | 銘柄あたり2回まで |

### 期待される効果（前場限定）

| 指標 | 値 |
|------|-----|
| 取引数 | 15件/4日 |
| 勝率 | 20% |
| PF | 2.52 |
| 純損益 | +49,959円/4日 |
| 期待値 | +3,330円/回 |

### 追加検討事項

1. **後場エントリー禁止**: 後場は全敗のため、前場限定が安全
2. **1日最大回数制限**: 同じ大台で複数回エントリーを避ける（6/30の11600円で3連敗）
3. **対象銘柄拡大**: 他の銘柄（8035, 6857等）でも同様のパターンがあるか検証が必要
4. **サンプル数不足**: 4日間19件は統計的に不十分。最低1ヶ月の検証を推奨

## 10. 結論

| 質問 | 回答 |
|------|------|
| sell_pressure時LONGは有効か？ | **条件付きで有効**（前場 + 大台超え + VWAP上） |
| 6/30をプラスにできるか？ | **前場限定なら可能**（+4,834円） |
| 即実装すべきか？ | **通知モードで1週間観察を推奨** |
| リスクは？ | 下落日に1件-9,749円の損失あり。BEストップで多くは保護される |

**最終判断**: PF 1.29（全時間帯）は手数料考慮後にマイナスリスクあり。前場限定（PF 2.52）であれば実装価値あり。ただしサンプル数4日間は統計的に不十分であり、まずは通知のみモードでの観察を推奨する。
