# MA25→MA20 + パーフェクトオーダー4要素化 比較バックテスト報告

**作成日**: 2026年6月27日  
**分析期間**: 2025年11月4日 〜 2026年6月12日（139営業日）  
**データソース**: J-Quants 1分足データ（20銘柄）  

---

## 1. 検証概要

提案された仕様変更が現行システムの損益にどのような影響を与えるかを、同一データで比較検証した。

| 比較対象 | 内容 |
|----------|------|
| **A: 現行仕様** | MA25 + 3要素信頼度（出来高・トレンド・勢い）+ ウォームアップ10本 |
| **B: MA20変更** | MA25→MA20に変更（他は同一）+ ウォームアップ10本 |
| **C: ウォームアップ100本** | 現行MA25 + 10:40以降のみエントリー（100本相当の効果測定） |
| **PO分析** | 現行エントリーをパーフェクトオーダー成立/非成立で分類し効果測定 |

## 3. 各変更の影響分析

### 3.1 MA25→MA20の影響

> **結論: MA20への変更は損益を38.5%悪化させる。採用非推奨。**

| 指標 | 差分（B - A） |
|------|:---:|
| 総損益 | **-257,300円 (-38.5%)** |
| 取引数 | +23件（増加） |
| 勝率 | -2.1ポイント |
| 最大DD | +28,400円（悪化） |

MA20はMA25より短期のため、クロスが頻発しダマシが増加する。取引回数は23件増えたが、勝率が2.1ポイント低下し、結果として大幅な損益悪化となった。特に主力銘柄の村田製作所で-173,700円、太陽誘電で-100,800円と大きく悪化している。

**銘柄別影響（データのある銘柄のみ）:**

| 銘柄 | A: MA25 | B: MA20 | 差分 |
|------|---:|---:|---:|
| 村田製作所 | +302,800円 | +129,100円 | **-173,700円** |
| 太陽誘電 | +358,900円 | +258,100円 | **-100,800円** |
| キオクシアHD | +7,000円 | +24,200円 | +17,200円 |

MA20が有利だったのはキオクシアHDのみ（+17,200円）で、主力2銘柄で合計-274,500円の悪化。MA25の方が1分足デイトレードに適した期間であることが確認された。

### 3.2 ウォームアップ100本（10:40以降のみ）の影響

> **結論: ウォームアップ100本は損益を73.7%悪化させる。採用非推奨。**

| 指標 | 差分（C - A） |
|------|:---:|
| 総損益 | **-492,900円 (-73.7%)** |
| 除外取引数 | 203件（全401件の50.6%） |
| 除外取引の損益 | **+492,900円**（利益の大半を占める） |

10:40以前のエントリーが全利益の73.7%を生み出している。これは寄り付き後の高ボラティリティ時間帯（9:00〜10:40）がデイトレードの最大の収益源であることを示している。ウォームアップを100本に延長すると、この最も収益性の高い時間帯を完全に放棄することになる。

**補足**: 現行のウォームアップ10本（約10分、9:10頃からシグナル有効）は、寄り付き直後の極端なノイズを避けつつ、高ボラ時間帯の機会を捕捉するバランスの取れた設定である。

### 3.3 パーフェクトオーダーの効果

> **結論: パーフェクトオーダーは信頼度の裏付けとして機能しない。むしろ若干不利。**

| 指標 | PO成立時 | PO非成立時 | 差分 |
|------|:---:|:---:|:---:|
| 該当件数 | 98件 (51.6%) | 92件 (48.4%) | — |
| 総損益 | +69,300円 | +83,800円 | **-14,500円** |
| 勝率 | 48.0% | 48.9% | **-0.9pt** |
| 平均損益/件 | +707円 | +911円 | **-204円** |

パーフェクトオーダーが成立している状態でのエントリーは、非成立時と比較して勝率が0.9ポイント低く、1件あたりの平均損益も204円低い。これは、パーフェクトオーダーが「トレンドが既に成熟した局面」を示すことが多く、**エントリータイミングとしてはむしろ遅い**（トレンドの終盤に乗ることになる）ためと考えられる。

## 4. 総合判定

| 提案項目 | 判定 | 理由 |
|----------|:---:|------|
| MA25→MA20 | **不採用** | -257,300円（-38.5%）の損益悪化。ダマシ増加。 |
| ウォームアップ100本 | **不採用** | -492,900円（-73.7%）。高収益時間帯を放棄。 |
| MA60/MA100追加計算 | **不要** | パーフェクトオーダーが無効のため計算コスト増のみ。 |
| パーフェクトオーダー4要素化 | **不採用** | 勝率-0.9pt、平均損益-204円/件。裏付けとして機能せず。 |
| strong+新設 | **不要** | 4要素目が無効のため、4点到達の意味がない。 |

---

## 5. 推奨事項

**現行仕様（MA25 + 3要素信頼度 + ウォームアップ10本）を維持する**ことを推奨する。

ただし、信頼度判定の強化を目指す場合は、以下の代替案を検討する価値がある:

1. **板読みスコアを4つ目の裏付けに追加**（既にデータ取得済み）
   - 板の売り圧力/買い圧力をシグナル方向と照合
   - パーフェクトオーダーと異なり「リアルタイムの需給」を反映するため、タイミング精度が高い可能性

2. **出来高急増の閾値を段階化**（現行1.2倍を1.5倍/2.0倍で追加ポイント）
   - 出来高が平均の2倍以上なら「強い裏付け」として2点付与
   - 既存フレームワーク内で実装可能

3. **ウォームアップは現行10本のまま、MA60のみ追加**（MA100は不要）
   - MA60は60本（約1時間）で計算可能。ウォームアップ拡大不要
   - 「MA5 < MA25 < MA60」の3段階パーフェクトオーダーなら10:00頃から判定可能
   - ただし効果は限定的（上記分析の示唆から）

---

## 6. 注意事項

本分析はJ-Quantsの過去データ（139営業日）に基づくバックテストであり、KABUステーションのリアルタイムデータとは若干の差異がある可能性がある。ただし、MA期間変更やウォームアップ延長の影響は構造的なものであり、データソースの違いで結論が逆転する可能性は低い。
