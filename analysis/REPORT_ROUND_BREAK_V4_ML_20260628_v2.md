## 5. 推奨: 候補A「板スコア>=2 + 買い圧力<0.54 + 13時前」

### 理由

1. **統計的根拠が最も強い**: 3つの有意変数を全て使用
2. **件数バランス**: 82件（通過率40.8%）は過学習リスクが低い
3. **ブロック効果**: 119件をブロック → 回避損失 **+220,350円**
4. **勝率50%**: 現行33.8%から大幅改善
5. **P&L +207,300円**: 現行-13,050円から**+220,350円の改善**

### 条件の解釈

```
① 板読みスコア >= 2
   → 売り板圧力が確認されている（sell_pressure + 買い圧力低 or 大口売り壁）
   → 板スコア0-1は全面的に負け（-139,050円）を回避

② 買い圧力比率 < 0.54
   → 買い板合計/売り板合計 が0.54未満（売り板が買い板の1.85倍以上）
   → 0.54以上は中立〜買い優勢で、大台割れが「ダマシ」になりやすい

③ 13時前（timeMinutes < 780）
   → 13時以降は勝率13.5%で壊滅的
   → 午後は反発・引け買いの影響で大台割れが機能しない
```

### 実装時の注意

- 買い圧力比率は`boardSnapshot.buyPressureRatio`から取得（既存フィールド）
- 板読みスコアは既存の`getBoardScore()`関数で算出済み
- 時間フィルターは`candleTime`から判定可能
- **既存の他フィルター（ATR率、強い上昇トレンドブロック等）は維持**
