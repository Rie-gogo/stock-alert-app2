### 2.4 信頼度（confidence）判定

3つの独立した裏付けをスコアリング:

| 裏付け | 条件 |
|--------|------|
| 出来高確認 | 当該足の出来高 ≥ 直近10本平均 × 1.2倍 |
| トレンド一致 | 買い: MA5≥MA25（またはレジーム=up）、売り: MA5≤MA25（またはレジーム=down） |
| 勢い一致 | 買い: 3本前からの価格差>0、売り: <0 |

| スコア | 信頼度 | 処理 |
|--------|--------|------|
| 3 | strong | シグナル生成 |
| 2 | medium | シグナル生成 |
| 0〜1 | weak | **シグナル非生成（抑制）** |

## 3. エントリー条件（realtimeSimEngine内）

### 3.1 共通フィルター（エントリー前に適用）

| フィルター | 条件 | 備考 |
|-----------|------|------|
| VWAPクロス上抜け無効化 | シグナル理由に「VWAPクロス上抜け」含む→ブロック | 5日間検証で0勝4敗のため |
| 板圧力逆方向チェック | 買い時にsell_pressure→ブロック、売り時にbuy_pressure→ブロック | |
| 板読みスコア | スコア < 1 → ブロック | 7要素統合スコア |
| HybridAフィルター | 始値比+0.2%以上（BULLISH相場）→SHORT禁止 | 売りのみ |
| **medium直接エントリー禁止** | **medium品質のステートマシン未経由シグナル→ブロック** | **改良策3改** |

### 3.2 板読みスコア（7要素、合計±10点）

| 要素 | 内容 | 配点 |
|------|------|------|
| A | アグレッシブ注文検出（marketOrderRatio≥0.08） | ±2 |
| B | 厚い板のアノマリー（largeBuyWall/largeSellWall） | ±1 |
| C | 板圧力トレンド（直近5本のBPR変化量≥0.15） | ±1 |
| D | 相場モード判定（active/building→+1、trap/quiet→-2） | +1/-2 |
| E | 板圧力の強さ（BPR≥1.4 or ≤0.65） | ±1 |
| F | 歩み値方向推定（uptick/downtick判定）【改良案B】 | ±2 |
| G | アイスバーグ/見せ板検出【改良案C】 | ±1 |

**閾値:** スコア ≥ 1 でエントリー許可（板情報なしの場合はスコア=1で中立通過）

### 3.3 ステートマシン経由エントリー

#### ダウ理論（押し目確認）ステートマシン

| ステップ | 条件 |
|---------|------|
| トリガー | 「ダウ理論: 直近高値更新」シグナル |
| 追加フィルター | 5分足MA5>MA25（上位足上昇トレンド確認） |
| 追加フィルター | 押し目深さ30〜70%の範囲内（直近20本基準） |
| 待機 | 最大5本（5分間） |
| 押し目確認 | 一度終値がシグナル価格を下回る |
| エントリー | 再上昇（終値>シグナル価格）でエントリー |
| キャンセル条件 | 直近安値割れ or タイムアウト(5本超過) |
| エントリー時チェック | 板読みスコア≥1、sell_pressure時はブロック |

#### 大台確認バーステートマシン

| ステップ | 条件 |
|---------|------|
| トリガー | 「大台超え/割れ」シグナル |
| 確認 | キリ番の上/下を5本連続維持 |
| 押し目待ち | 確認完了後、さらに最大5本の押し目待ち |
| エントリー | 押し目確認後にエントリー、またはタイムアウト時（強トレンド）にエントリー |
| キャンセル条件 | キリ番割れ |
| エントリー時チェック | 板読みスコア≥1、逆方向板圧力時はブロック |

### 3.4 直接エントリー

ステートマシントリガー（ダウ理論・大台）以外のシグナルは直接エントリー:
- **strong品質のみ許可**（改良策3改: medium品質はブロック）
- 板読みスコア≥1
- 板圧力逆方向チェック通過
- ATRフィルター通過
- 証拠金使用率制限通過

## 4. イグジット条件

### 4.1 損切り（Stop Loss）

| 項目 | 値 |
|------|-----|
| 損切り率 | **0.5%** |
| 判定 | LONG: 安値 ≤ エントリー価格×(1-0.5%) / SHORT: 高値 ≥ エントリー価格×(1+0.5%) |
| 約定価格 | 損切りライン（スリッページなし） |

### 4.2 利確（Take Profit）

| 項目 | 値 |
|------|-----|
| 利確率 | **1.5%** |
| 判定 | LONG: 高値 ≥ エントリー価格×(1+1.5%) / SHORT: 安値 ≤ エントリー価格×(1-1.5%) |
| 約定価格 | 利確ライン（スリッページなし） |

### 4.3 シグナル反転決済

- LONG保有中に売りシグナル発生 → 終値で決済
- SHORT保有中に買いシグナル発生 → 終値で決済

### 4.4 板読み早期利確

| 条件 | 内容 |
|------|------|
| 最低利益率 | 0.05%以上の含み益があること |
| LONG | sell_pressure or large_sell_wall検出 → 終値で利確 |
| SHORT | buy_pressure or large_buy_wall検出 → 終値で利確 |

### 4.5 大引け強制決済

- 15:30の足で全オープンポジションを終値で強制決済

### 4.6 イグジット優先順位

1. 損切り（最優先）
2. 利確
3. シグナル反転決済
4. 板読み早期利確
5. 大引け強制決済

## 5. レジーム（大局トレンド）判定

### 5.1 判定材料

| 材料 | 計算方法 | 閾値 |
|------|---------|------|
| MA25の傾き | (現在MA25 - 10本前MA25) / 10本前MA25 | ±0.15% |
| 当日騰落率 | (終値 - 当日始値) / 当日始値 | ±0.6% |

### 5.2 判定ルール

| レジーム | 条件 |
|---------|------|
| down | 下落材料≥1 かつ 上昇材料=0 |
| up | 上昇材料≥1 かつ 下落材料=0 |
| neutral | それ以外 |

### 5.3 シグナル許可ルール

| レジーム | 買い | 売り |
|---------|------|------|
| down | **抑制** | 許可 |
| up | 許可 | **抑制** |
| neutral | 許可 | 許可 |
