# 9の法則（5分足ベース）バックテスト報告

**作成日**: 2026年6月27日  
**分析期間**: 2025年11月4日 〜 2026年6月12日（139営業日）  
**データソース**: J-Quants 1分足データ（17銘柄）  

---

## 1. 検証概要

「5分足で9本（=45分）以上保有で含み益があれば利確」「5分足で5本（=25分）以上保有で膠着なら撤退」を現行エグジットに追加した場合の損益影響を検証。

### テスト仕様

| ルール | 条件（5分足ベース） | 1分足換算 | アクション |
|--------|------|------|-----------|
| **ルールA** | 9本以上保有 + 含み益 > 0 + 含み益 < 1% | 45本（45分） | 利確 |
| **ルールB** | 5本以上保有 + \|損益率\| < 0.1% | 25本（25分） | 撤退 |

**重要**: 含み益 ≥ 1%（トレイリング圏内）の場合は9の法則を適用せず、トレイリングに委ねる設計。

---

---

## 3. 判定: 微減だがリスク改善あり

5分足ベースでは1分足ベース（-68.3%）と比較して影響は大幅に縮小し、**-51,000円（-7.6%）の微減**に留まった。一方で:

- **勝率は+0.3pt改善**（51.1%→51.4%）
- **負け日が2日減少**（48日→46日）
- **最大DDが-6,100円改善**（49,900円→43,800円）

収益性は微減だが、リスク管理面では改善が見られる。

---

## 4. ルール別詳細分析

### 4.1 ルールA（45分含み益利確）

| 指標 | 値 |
|------|------|
| 発火回数 | **14回**（全401取引の3.5%） |
| 9の法則での損益 | +129,300円（平均+9,236円/回） |
| 現行での同取引損益 | +137,100円（平均+9,793円/回） |
| **差分** | **-7,800円** |
| 現行より改善した件数 | 7/14 (50.0%) |

ルールAは発火頻度が非常に低く（14回のみ）、影響は限定的。含み益<1%の条件によりトレイリング圏内の取引を保護しているため、大きな利益の取り逃しは発生していない。**ちょうど半分（50%）のケースで現行より改善**しており、コイントスに近い。

### 4.2 ルールB（25分膠着撤退）

| 指標 | 値 |
|------|------|
| 発火回数 | **75回**（全401取引の18.7%） |
| 9の法則での損益 | +3,300円（平均+44円/回） |
| 現行での同取引損益 | +46,500円（平均+620円/回） |
| **差分** | **-43,200円** |
| 現行より改善した件数 | 32/75 (42.7%) |

ルールBが主な悪化要因。75回発火のうち42.7%（32件）で改善したが、残りの57.3%（43件）は「膠着に見えたが、その後動き出して利益になった」ケース。

> **核心的な問題**: 25分間の膠着（±0.1%以内）は「まだ方向感が出ていない」だけで、その後のブレイクで利益になるケースが多い。現行ではこれらの取引が平均+620円/回の利益を生んでいるが、ルールBで撤退すると平均+44円/回に激減する。

## 6. ルールA単独採用の検討

ルールBが主な悪化要因であることが判明したため、**ルールA単独**の場合を評価する:

| 指標 | ルールA単独 |
|------|------|
| 影響取引数 | 14件のみ |
| 損益差分 | -7,800円 |
| 改善率 | 50.0% |
| 最大DD影響 | ほぼなし |

ルールA単独なら**-7,800円（-1.2%）のほぼ無影響**。ただし改善率50%（コイントス）であり、積極的に採用する根拠も薄い。

---

## 7. 結論と推奨

| 構成 | 損益差分 | DD改善 | 判定 |
|------|:---:|:---:|:---:|
| ルールA + ルールB | -51,000円 (-7.6%) | -6,100円 | **条件付き採用可** |
| ルールA単独 | -7,800円 (-1.2%) | 微小 | **ほぼ無影響** |
| ルールB単独 | -43,200円 (-6.5%) | -6,100円 | **不採用** |

### 推奨判断

**ルールA（45分含み益利確）は採用してもほぼ無害**だが、積極的なメリットも薄い（50%コイントス）。

**ルールB（25分膠着撤退）は不採用を推奨**。25分の膠着後に動き出すケースが多く、早期撤退は逸失利益を生む。

### 改善案（ルールBを改善する場合）

| パラメータ | 現提案 | 改善案 | 理由 |
|------------|--------|--------|------|
| 膠着判定本数 | 25本(25分) | **40本(40分)** | 「本当の膠着」のみ検出 |
| 膠着幅 | ±0.1% | **±0.05%** | より厳密な判定 |
| 追加条件 | なし | **出来高減少** | 「枯れた」状態のみ |

40分以上かつ±0.05%以内かつ出来高減少 → これなら「本当に動かない」ケースのみ撤退でき、逸失利益を最小化できる可能性がある。
