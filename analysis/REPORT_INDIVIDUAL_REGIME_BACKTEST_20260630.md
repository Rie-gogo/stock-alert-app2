**作成日**: 2026年6月30日  
**検証期間**: 2026-06-17 〜 2026-06-30（10営業日）  
**対象銘柄**: 6920, 6857, 5803, 6976, 6981, 6526, 9984, 7011, 8035, 8316（10銘柄）

## 1. 概要

各銘柄ごとにリアルタイムでレジーム（5段階）を判定し、mediumシグナルの許可/禁止を制御する方式の有効性を検証した。

### レジーム5段階

| レジーム | 判定条件 |
|----------|----------|
| STRONG_BULL | price > VWAP AND MA5 > MA25 AND MA25傾き > 0 AND 直近15分高値更新 |
| WEAK_BULL | price > VWAP AND MA5 > MA25 |
| RANGE | abs(price-VWAP)/VWAP < 0.3% AND abs(MA5-MA25)/MA25 < 0.3% |
| WEAK_BEAR | price < VWAP AND MA5 < MA25 |
| STRONG_BEAR | price < VWAP AND MA5 < MA25 AND MA25傾き < 0 AND 直近15分安値更新 |

### mediumシグナルへの適用ルール

| レジーム | SHORT medium | LONG medium |
|----------|-------------|-------------|
| STRONG_BULL | 禁止 | 許可 |
| WEAK_BULL | 禁止 | 許可 |
| RANGE | 板スコア閾値+1 | 板スコア閾値+1 |
| WEAK_BEAR | 許可 | 禁止 |
| STRONG_BEAR | 許可 | 禁止 |

### 4パターン定義

| パターン | 説明 |
|----------|------|
| A | 現行（medium全ブロック） |
| B | SHORT medium解除（レジームなし） |
| C | 個別銘柄レジームのみ追加（mediumブロック維持、レジームで一部解除） |
| D | 個別銘柄レジーム + SHORT medium解除 |

### 既存フィルター（全パターン共通）

- VWAP急落フィルター（5本-0.8% or 3本-0.6%）
- BEストップ（+0.5%トリガー）
- 後場大台割れSHORT BPR>=0.65ブロック
- 時間帯フィルター（11:00-11:30, 12:30-13:00除外）
- SL 0.5%, TP 1.5%

## 10. 結論と採用判定

### 判定基準: PF >= 1.15 かつ 最大DD <= 200,000円

| パターン | PF | 最大DD | 判定 |
|----------|-----|--------|------|
| A: 現行 | 1.01 | 248,859円 | **不採用**（PF不足 + DD超過） |
| B: SHORT medium解除 | **1.17** | 229,421円 | **DD注意**（PF基準クリア、DD超過） |
| C: レジームのみ | 0.89 | 392,228円 | **不採用**（PF < 1.0、大幅マイナス） |
| D: レジーム+SHORT解除 | 0.93 | 416,386円 | **不採用**（PF < 1.0、大幅マイナス） |

### 最終結論

> **個別銘柄リアルタイムレジーム方式（パターンC/D）は不採用。**

理由:
1. **レジームによるmedium解除が逆効果**: WEAK_BULL/WEAK_BEARでのmedium許可が大量のダマシを生み、PF < 1.0（損失発生）
2. **LONG medium解除が特に有害**: パターンC/DのLONGは-217,316円/-187,412円と大幅マイナス
3. **6/29の大損失**: レジームがWEAK_BEARと判定してmediumを許可した結果、-188,505円の損失
4. **最大DDが400,000円超**: 許容範囲（200,000円）の2倍

### パターンBの評価

パターンB（SHORT medium解除、レジームなし）は:
- PF 1.17（基準クリア）
- 総損益 +506,671円（最高）
- 下落日で+746,714円の大幅利益
- ただし最大DD 229,421円（基準超過）
- 上昇日に-240,043円の損失

**Bは「上昇日のSHORT medium抑制」を追加すれば採用可能性あり**。これは前回検証のC2（固定上昇日判定）と同じ方向性。

### 推奨次ステップ

1. **パターンBベースで上昇日フィルターを追加**: 寄り付き30分の値動きで当日の方向性を暫定判定し、上昇傾向ならSHORT mediumをブロック
2. **前場エントリー抑制の強化**: 全パターンで前場がマイナスのため、前場のmediumエントリーのみ追加抑制を検討
3. **6981村田製作所の除外検討**: 全パターンで大幅マイナス（-85,568円〜-119,798円）
