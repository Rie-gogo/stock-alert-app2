
## 教訓まとめ

1. **上昇日にSHORTのみ = 構造的に負ける**: 6/30は全体が上昇相場だったが、システムはSHORTしか入れなかった

2. **板データと値動きの乖離**: sell_pressureが出ていても「売り板を突破する買い」が入れば上昇する。現在のシステムはこのパターンを認識できない

3. **VWAPクロス下抜けの急落局面リスク**: 急落中にVWAP下抜けシグナルが出ると、底値でSHORTする逆張りになりやすい（9984の2分損切りが典型）

4. **BUYの通過率が構造的に低い**: 同じ市場環境でもSELLは3.9%通過するのにBUYは0.7%しか通過しない。sell_pressureがLONGにとって二重の障壁（即ブロック + スコア低下）になっている

5. **後場のBUYシグナルは高値掴みリスク**: 仮にフィルターを緩和しても、後場のBUYは利益確定売りに巻き込まれる可能性がある。前場の早い段階でBUYに入る仕組みが必要

---

## 次回への改善提案

1. **売り板突破パターンの認識**: sell_pressureでも価格が上昇している場合、LONGを許可するロジックの追加
2. **朝の地合い判定**: 日経先物や寄り付きの動きから上昇日/下落日を早期判定し、方向性を合わせる
3. **VWAPクロス下抜けの急落フィルター**: 直近N本で急落中（例: 5本で-1%以上）の場合はVWAP下抜けSHORTをブロック
