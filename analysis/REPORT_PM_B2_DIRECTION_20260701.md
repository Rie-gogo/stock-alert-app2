## 1. シミュレーション概要

現行エンジンでは、B2方式の地合い判定は**前場のみ**（9:30時点で判定）に適用され、後場のSHORT mediumは無条件で許可されている。本シミュレーションでは、後場にも同様の地合い判定（PM_B2_DIRECTION）を導入した場合の損益影響を検証する。

### 後場地合い判定ロジック

後場B2方式は、12:30時点（後場開始直後）の全銘柄の「始値→12:30終値」の変動率平均で判定する。

| 条件 | 判定結果 |
|------|----------|
| 平均変動率 >= +0.2% | **bullish** |
| 平均変動率 <= -0.2% | **bearish** |
| それ以外 | **neutral** |

### 比較パターン

| パターン | 後場SHORT medium処理 | 説明 |
|----------|---------------------|------|
| **A: 現行** | 無条件許可 | 後場はB2判定を適用しない（現行エンジン） |
| **B: bullish時ブロック** | bullish日のみブロック | 後場地合いがbullishの日はSHORT mediumを禁止 |
| **C: bullish+neutral時ブロック** | bearishのみ許可 | 後場地合いがbearishの日のみSHORT mediumを許可 |
| **D: 全ブロック** | 全面禁止 | 後場SHORT mediumを一切許可しない |

### 共通条件

- ロットサイズ: 270万円/銘柄
- 損切り(SL): 0.5%
- 利確(TP): 1.5%
- ブレイクイーブン(BE): +0.5%到達で発動
- 前場B2: 9:30固定判定（bullish時SHORT mediumブロック）
- BUY medium: 全ブロック
- 後場BPR>=0.65: 全SHORTブロック
- VWAP急落フィルター: 5本-0.8%/3本-0.6%

## 7. 結論と推奨

### 結論

後場B2方式（PM_B2_DIRECTION）の導入は、**いずれのパターンでも現行より損益が悪化する**結果となった。

1. **bullish日でもSHORT mediumは有効**: 後場がbullish判定の日でも、後場SHORT mediumは+166,778円（36.8%勝率）の利益を生んでいる。これは、個別銘柄レベルでは全体地合いと逆方向の動きが存在するためと考えられる。

2. **B と C が同一結果**: neutral日の後場SHORT mediumが0件だったため、bullishブロックとbullish+neutralブロックの結果が完全に一致した。neutral日のサンプルが1日しかないため、この結論は暫定的である。

3. **全ブロック(D)は大幅悪化**: 後場SHORT mediumを全面禁止すると、SHORT全体の損益がマイナスに転落し、総損益が約50万円減少する。

### 推奨

> **現行のまま（後場SHORT medium無条件許可）を維持することを推奨する。**

後場の地合い判定によるSHORT mediumブロックは、この10日間のデータでは利益を減少させる方向に作用している。後場は前場と異なり、地合いの方向性と個別銘柄のショート機会が必ずしも連動しないため、一律のブロックは不適切と考えられる。

ただし、サンプル期間が10日間と短いため、今後データが蓄積された段階で再検証することを推奨する。特にneutral日のサンプルが1日のみであり、より長期のデータで傾向を確認する必要がある。
