# BUYシグナル取り逃し原因調査レポート

**対象日**: 2026年6月30日（月）  
**調査目的**: 上昇銘柄（太陽誘電・フジクラ・レーザーテック・村田製作所等）でLONGエントリーチャンスがあったにもかかわらず、システムがBUYを拾えなかった原因の特定  
**データソース**: KABUステーション取得の1分足リアルタイムデータ + 板データ

---

## エグゼクティブサマリー

6月30日のBUYシグナルは全7銘柄で合計**207件**生成されたが、実エントリーに至ったのはわずか**3件**（通過率1.4%）であった。一方SELLシグナルは48件中4件がエントリー（通過率8.3%）であり、BUYの通過率はSELLの**約6分の1**に留まっている。

しかし、Ghost Trade検証（ブロックされたBUYを仮想的にエントリーした場合のシミュレーション）の結果は**-121,248円**の損失となり、**フィルターが全体としては正しく機能していた**ことが判明した。ただし銘柄別に見ると、村田製作所（+40,397円）とレーザーテック（+13,222円）では利益が出ており、フィルターの一律適用に改善余地がある。

## 実施4: sell_pressure時LONGブロック妥当性検証

### パターンA（現行: sell_pressure時BUY全ブロック）

- ブロック数: 45件（うちstrong 17件）
- Ghost Trade損益（strong 17件）: **-257円**（ほぼ収支均衡）

### パターンB（条件付き許可）

以下の全条件を同時に満たす場合のみ仮想LONG許可：
- close > VWAP
- 直近3本で陽線優勢（2/3以上）
- 直近3本累積上昇率 >= 1.2%
- BPRが3本前より+0.05以上改善
- volumeRatio >= 1.5
- strongシグナルのみ

**結果: 条件通過数 = 0件**

全5条件を同時に満たすケースは6/30には存在しなかった。特に「BPR+0.05改善」と「volumeRatio >= 1.5」の同時充足が困難であった。

### A/B比較結論

| 比較項目 | A（全ブロック） | B（条件付き許可） |
|---------|---------------|-----------------|
| 仮想取引数 | 17件 | 0件 |
| Ghost損益 | -257円 | 0円 |
| 差分 | - | +257円 |

条件Bの5条件は厳しすぎて実質的に機能しない。sell_pressureブロック自体のGhost損益が-257円とほぼゼロであることから、**sell_pressure_blockは「過剰」ではなく「やや保守的」**と評価できる。

## 実施5: 最終結論

### 1. BUYが入れなかった最大原因

**time_filter（57件, 27.9%）** が最多だが、これは市場構造上の制約。実質的な最大原因は**sell_pressure_block（45件, 22.1%）**と**state_machine_waiting（39件, 19.1%）**の複合効果である。

### 2. sell_pressure_blockは過剰か

**結論: 過剰ではない。** Ghost Trade損益が-257円とほぼ収支均衡であり、ブロックしてもしなくても全体損益に大きな影響はない。ただし**銘柄別に見ると村田製作所（+80,799円）とレーザーテック（+40,095円）では大きな利益機会を逃している**一方、フジクラ（-40,466円）と太陽誘電（-107,608円）では正しくブロックしている。

### 3. board_score_lowは過剰か

**結論: 過剰ではない。** 6/30ではわずか1件のみのブロックであり、問題にならない。

### 4. state_machine_waitingでチャンスを逃しているか

**結論: 逃していない。** Ghost Trade全19件が0%勝率（-120,992円）であり、確認ステートマシンが正しく「偽ブレイクアウト」を除外している。太陽誘電の大台超えシグナルが頻発するが、確認待機中にBE到達→建値戻しのパターンが多く、直接エントリーしていれば大損していた。

### 5. 今日のチャート上の上昇をシステムが取れなかった理由

6/30の上昇は「sell_pressure環境下での力強い買い突破」パターンであった。システムはsell_pressure時にLONGを一律禁止するため、この種の上昇を構造的に取れない。特に：

- **村田製作所**: 10:15に11000円突破、10:53に11200円突破 → 両方sell_pressureでブロック → 両方TP到達（+80,799円の機会損失）
- **レーザーテック**: 10:14に49500円突破 → sell_pressureでブロック → TP到達（+40,095円の機会損失）

### 6. フィルターを緩和すべきか

**結論: 一律緩和は危険。銘柄選択的な緩和を検討すべき。**

Ghost Trade全体では-121,248円の損失であり、フィルターを一律緩和すると損失が拡大する。しかし銘柄別に見ると明確な二極化がある：

| 銘柄群 | Ghost PnL | 緩和の方向性 |
|--------|-----------|-------------|
| 村田製作所・レーザーテック | +53,619円 | 緩和候補 |
| 太陽誘電・フジクラ・アドバンテスト | -174,867円 | 現行維持 |

### 7. 緩和するならどの条件をどの順番で

**推奨緩和順序:**

1. **sell_pressure_block の銘柄別緩和**（優先度: 高）  
   村田製作所・レーザーテックに限定して、sell_pressure時でも大台超えシグナル（strong）のLONGを許可する。条件として「直近3本陽線優勢 + VWAP上」を付加。

2. **state_machine_waitingの即時エントリーモード検討**（優先度: 低）  
   現状は正しく機能しており、緩和は不要。太陽誘電のBE多発問題は、BEトリガーを+0.7%に引き上げることで対処可能だが、他銘柄への影響を要検証。

3. **time_filter**（優先度: なし）  
   市場構造上の制約であり、緩和不可。
